რეკომენდირებული:

  • პროცესორი: 2 გჰც, 2 ბირთვი
  • ოპერატიული მეხსიერება: 4 გბ
  • დისკის ადგილი: 32 გბ
  • გრაფიკა: 1024x768

მინიმალური:

  • პროცესორი: 1 გჰც
  • ოპერატიული მეხსიერება: 2 გბ
  • დისკის ადგილი: 4 გბ
1 ჩამტვირთავი მედიის შექმნა

ჩაწერეთ B1ackOS-ის სურათი USB დისკზე Rufus-ის მსგავსი პროგრამების გამოყენებით.

2 BIOS/UEFI კონფიგურაცია

გადატვირთეთ კომპიუტერი და დააკონფიგურირეთ ჩატვირთვა USB-დან BIOS/UEFI-ში.

3 ინსტალაციის დაწყება

მიჰყევით ინსტალატორის ინსტრუქციებს, აირჩიეთ დანაყოფი და დააკონფიგურირეთ სისტემის პარამეტრები.

4 ინსტალაციის დასრულება

ინსტალაციის დასრულების შემდეგ სისტემა გადაიტვირთება. შექმენით ანგარიში და დააკონფიგურირეთ პარამეტრები.

გაფრთხილება: ინსტალაციამდე დარწმუნდით, რომ თქვენი აპარატურა აკმაყოფილებს მინიმალურ მოთხოვნებს. რეკომენდირებულია მნიშვნელოვანი მონაცემების სარეზერვო ასლის შექმნა.

სამუშაო გარემო

Cinnamon

თანამედროვე, ინოვაციური
Cinnamon არის თანამედროვე სამუშაო გარემო, რომელიც გვთავაზობს კლასიკურ ინტერფეისს ინოვაციური ფუნქციებით. ის უზრუნველყოფს გლუვ მუშაობას და მაღალი ხარისხის პერსონალიზაციას.
ჩამოტვირთვა
sudo apt install -y cinnamon

GNOME

მარტივი, ელეგანტური, ინტუიტიური
GNOME ფოკუსირებულია სიმარტივესა და გამოყენების სიმარტივეზე. ის გთავაზობთ სუფთა და მინიმალისტურ ინტერფეისს.
sudo apt install -y gnome

KDE Plasma

ძლიერი, პერსონალიზირებადი, თანამედროვე
KDE Plasma გთავაზობთ ფუნქციების მდიდარ კომპლექტს და პერსონალიზაციის მაღალ ხარისხს. ეს არის ერთ-ერთი ყველაზე მოწინავე სამუშაო გარემო თანამედროვე დიზაინით.
sudo apt install -y sddm kde-plasma-desktop

Xfce

მსუბუქი, მარტივი, ეფექტური
Xfce არის მსუბუქი სამუშაო გარემო. ის არ უჭერს მხარს იმდენ ფუნქციას, რამდენსაც Cinnamon, მაგრამ ნაკლებ რესურსს მოიხმარს, რაც მას იდეალურს ხდის ძველი კომპიუტერებისთვის.
sudo apt install -y xfce4

MATE

ტრადიციული, სტაბილური, საიმედო
MATE აგრძელებს GNOME 2-ის ტრადიციებს, გვთავაზობს კლასიკურ ინტერფეისს, რომელიც ნაცნობია მრავალი მომხმარებლისთვის. ის სტაბილურია და მოიხმარს ზომიერ რესურსებს.
sudo apt install -y slim mate
