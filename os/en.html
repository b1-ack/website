1 Create bootable media

Write the B1ackOS image to a USB drive using programs like Rufus.

2 BIOS/UEFI Setup

Restart your computer and configure boot from USB in BIOS/UEFI.

3 Start Installation

Follow the installer instructions, select the installation partition and configure system settings.

4 Complete Installation

After installation is complete, the system will reboot. Set up your account and settings as desired.

Desktop Environments

Cinnamon

Sleek, modern, innovative
Cinnamon is a modern desktop environment that offers a classic interface with innovative features. It provides smooth performance and a high degree of customization.
Download
sudo apt install -y cinnamon

GNOME

Simple, elegant, intuitive
GNOME focuses on simplicity and ease of use. It offers a clean and minimalist interface.
sudo apt install -y gnome

KDE Plasma

Powerful, customizable, modern
KDE Plasma offers a rich set of features and a high degree of customization. It's one of the most advanced desktop environments with a modern design.
sudo apt install -y sddm kde-plasma-desktop

Xfce

Light, simple, efficient
Xfce is a lightweight desktop environment. It doesn't support as many features as Cinnamon but it's lighter on resource usage, making it ideal for older computers.
sudo apt install -y xfce4

MATE

Traditional, stable, reliable
MATE continues the GNOME 2 tradition, offering a classic interface that is familiar to many users. It is stable and consumes a moderate amount of resources.
sudo apt install -y slim mate
