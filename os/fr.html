Télécharger Rufus

Recommandé :

  • CPU : 2 GHz, 2 cœurs
  • RAM : 4 Go
  • Espace disque : 32 Go
  • Graphisme : 1024x768

Minimum :

  • CPU : 1 GHz
  • RAM : 2 Go
  • Espace disque : 4 Go
1 Créer un support bootable

Écrivez l'image B1ackOS sur une clé USB avec des programmes comme Rufus.

2 Configuration BIOS/UEFI

Redémarrez votre ordinateur et configurez le démarrage sur USB dans le BIOS/UEFI.

3 Lancer l'installation

Suivez les instructions, sélectionnez la partition et configurez les paramètres système.

4 Terminer l'installation

Après l'installation, le système redémarrera. Configurez votre compte et vos paramètres selon vos préférences.

Attention : Avant l'installation, assurez-vous que votre matériel répond aux exigences minimales. Il est recommandé de sauvegarder vos données importantes.

Environnements de bureau

Cinnamon

Moderne, innovant
Cinnamon est un environnement de bureau moderne qui offre une interface classique avec des fonctionnalités innovantes. Il offre des performances fluides et un haut degré de personnalisation.
Télécharger
sudo apt install -y cinnamon

GNOME

Simple, élégant, intuitif
GNOME se concentre sur la simplicité et la facilité d'utilisation. Il offre une interface propre et minimaliste.
sudo apt install -y gnome

KDE Plasma

Puissant, personnalisable, moderne
KDE Plasma offre un riche ensemble de fonctionnalités et un haut degré de personnalisation. C'est l'un des environnements de bureau les plus avancés avec un design moderne.
sudo apt install -y sddm kde-plasma-desktop

Xfce

Léger, simple, efficace
Xfce est un environnement de bureau léger. Il ne supporte pas autant de fonctionnalités que Cinnamon mais consomme moins de ressources, ce qui le rend idéal pour les anciens ordinateurs.
sudo apt install -y xfce4

MATE

Traditionnel, stable, fiable
MATE perpétue la tradition de GNOME 2, offrant une interface classique familière à de nombreux utilisateurs. Il est stable et consomme une quantité modérée de ressources.
sudo apt install -y slim mate
