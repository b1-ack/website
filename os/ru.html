Рекомендуемые:

  • Процессор: 2 ГГц, 2 ядра
  • Оперативная память: 4 ГБ
  • Место на диске: 32 ГБ
  • Видеокарта: 1024x768

Минимальные:

  • Процессор: 1 ГГц
  • Оперативная память: 2 ГБ
  • Место на диске: 4 ГБ
1 Создание загрузочного носителя

Запишите образ B1ackOS на USB-накопитель с помощью программ вроде Rufus.

2 Настройка BIOS/UEFI

Перезагрузите компьютер и настройте загрузку с USB-носителя в BIOS/UEFI.

3 Запуск установки

Следуйте инструкциям установщика, выберите раздел для установки и настройте параметры системы.

4 Завершение установки

После завершения установки система перезагрузится. Настройте учетную запись и параметры по своему усмотрению.

Внимание: Перед установкой убедитесь, что ваше оборудование соответствует минимальным системным требованиям. Рекомендуется создать резервную копию важных данных.

Окружения рабочего стола

Cinnamon

Современный, инновационный
Cinnamon — это современная среда рабочего стола, которая предлагает классический интерфейс с инновационными функциями. Она обеспечивает плавную работу и высокую степень настройки.
Скачать
sudo apt install -y cinnamon

GNOME

Простой, элегантный, интуитивный
GNOME фокусируется на простоте и удобстве использования. Он предлагает чистый и минималистичный интерфейс.
sudo apt install -y gnome

KDE Plasma

Мощный, настраиваемый, современный
KDE Plasma предлагает богатый набор функций и высокую степень настройки. Это одна из самых продвинутых сред рабочего стола с современным дизайном.
sudo apt install -y sddm kde-plasma-desktop

Xfce

Легкий, простой, эффективный
Xfce — это легковесная среда рабочего стола. Она не поддерживает столько функций, как Cinnamon, но потребляет меньше ресурсов, что делает ее идеальной для старых компьютеров.
sudo apt install -y xfce4

MATE

Традиционный, стабильный, надежный
MATE продолжает традиции GNOME 2, предлагая классический интерфейс, который знаком многим пользователям. Он стабилен и потребляет умеренное количество ресурсов.
sudo apt install -y slim mate
