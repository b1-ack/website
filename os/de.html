Rufus herunterladen

Empfohlen:

  • CPU: 2 GHz, 2 Kerne
  • RAM: 4 GB
  • Festplattenspeicher: 32 GB
  • Grafik: 1024x768

Minimum:

  • CPU: 1 GHz
  • RAM: 2 GB
  • Festplattenspeicher: 4 GB
1 Boot-Medium erstellen

Schreiben Sie das B1ackOS-Image mit Programmen wie Rufus auf einen USB-Stick.

2 BIOS/UEFI einrichten

Starten Sie Ihren Computer neu und konfigurieren Sie den Boot von USB im BIOS/UEFI.

3 Installation starten

Folgen Sie der Installationsanleitung, wählen Sie die Partition aus und konfigurieren Sie die Systemeinstellungen.

4 Installation abschließen

Nach der Installation wird das System neu gestartet. Richten Sie Ihr Konto und Ihre Einstellungen nach Wunsch ein.

Achtung: Stellen Sie vor der Installation sicher, dass Ihre Hardware die Mindestanforderungen erfüllt. Es wird empfohlen, ein Backup Ihrer wichtigen Daten zu erstellen.

Desktop-Umgebungen

Cinnamon

Modern, innovativ
Cinnamon ist eine moderne Desktop-Umgebung, die eine klassische Oberfläche mit innovativen Funktionen bietet. Sie sorgt für eine flüssige Leistung und ein hohes Maß an Anpassbarkeit.
Herunterladen
sudo apt install -y cinnamon

GNOME

Einfach, elegant, intuitiv
GNOME konzentriert sich auf Einfachheit und Benutzerfreundlichkeit. Es bietet eine saubere und minimalistische Oberfläche.
sudo apt install -y gnome

KDE Plasma

Leistungsstark, anpassbar, modern
KDE Plasma bietet eine Vielzahl von Funktionen und ein hohes Maß an Anpassbarkeit. Es ist eine der fortschrittlichsten Desktop-Umgebungen mit modernem Design.
sudo apt install -y sddm kde-plasma-desktop

Xfce

Leicht, einfach, effizient
Xfce ist eine leichtgewichtige Desktop-Umgebung. Sie unterstützt nicht so viele Funktionen wie Cinnamon, verbraucht aber weniger Ressourcen, was sie ideal für ältere Computer macht.
sudo apt install -y xfce4

MATE

Traditionell, stabil, zuverlässig
MATE führt die GNOME-2-Tradition fort und bietet eine klassische Oberfläche, die vielen Benutzern vertraut ist. Es ist stabil und verbraucht moderate Ressourcen.
sudo apt install -y slim mate
